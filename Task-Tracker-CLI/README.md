## Task Tracker

Task Tracker is a program that allows you to track your tasks and centralize your activities, monitor progress and stay productive.

## 📦 Installation

```bash
git clone https://github.com/hamoudabass/Task-Tracker-CLI.git
```

## 🛠 Usage

```bash
python main.py
```

Example command or input/output:

```bash
1. Add a Task
2. Update a Task
3. Delete a Task
...
9. Quit
```

### Example task:

```bash

>>> Choose an option(1-9): 1

>>> Name of task: cooking a pizza

Task added successfully (ID: 1)

```
